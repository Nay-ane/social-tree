Anatomia de um documento HTML

< !DOCTYPE html >

Parte inicial obrigatória do documento. Basicamente garante que o documento se comporte corretamente.

< html >

Envolve todo o conteúdo da página conhecido como elemento raiz.

< head >

Age como um recipiente de tudo o que é necessário incluir em uma página HTML que não é conteúdo para ser mostrado. Ex: conjunto de caracteres, palavras-chaves, título.

< meta charset = "utf-8" >

Define o conjunto de caracteres que o documento deve usar para o utf-8, que inclui praticamente todos os caracteres da grande maioria dos idiomas escritos.

<  title >

Define o título da página, que é o título que aparece na guia do navegador onde a página é carregada.

<  body >

Contém todo o conteúdo que será mostrado aos visitantes da página, seja texto,imagens ou vídeos.
